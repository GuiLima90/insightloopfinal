# InsightLoop

## About the project

InsightLoop is a web application developed as the **final group project** during the **Le Wagon Full-Stack Web Development Bootcamp**.

The goal of the project is to analyze customer support conversations and extract insights to help businesses identify recurring issues and improvement opportunities.

This repository is a **fork of the original group project**, created for portfolio and learning purposes.

---

## Key Features

- User authentication
- Customer support conversation management
- Categorization of customer requests
- Insights and improvement suggestions
- Dashboard with aggregated data

---

## Tech Stack

- Ruby on Rails
- PostgreSQL
- HTML / SCSS
- JavaScript
- Bootstrap
- OpenAI API (AI-powered insights)

---

## Team Project

This project was developed collaboratively by a team during the Le Wagon bootcamp.  
Contributions include backend development, database modeling, UI implementation, and AI prompt design.
